# Quickstart: Console Todo Application

## Prerequisites
- Python 3.8 or higher
- No external dependencies required

## Setup
1. Clone or download the repository
2. Navigate to the project directory
3. Ensure Python 3.8+ is installed and accessible

## Running the Application
```bash
python todo_app/main.py
```

## Basic Usage
1. Run the application using the command above
2. The main menu will display with numbered options
3. Enter the number corresponding to the desired action
4. Follow the prompts to complete your task
5. The application will return to the main menu after each operation
6. Select option 6 to exit the application

## Example Workflow
1. Select "1. Add Task" to create a new task
2. Enter a title for your task
3. Optionally enter a description or press Enter to skip
4. Return to the main menu
5. Select "2. View Task List" to see your tasks
6. Continue using other options as needed
7. Select "6. Exit" when finished

## Error Handling
- If you enter invalid input, an error message will display
- The application will return to the main menu after the error
- No data is lost when errors occur
